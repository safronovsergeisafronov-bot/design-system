## Usage in Design System

All 287 icons are available in your Figma file in the following sizes:
- **16px** - For inline text, small buttons
- **20px** - For form inputs, small UI elements
- **24px** - For navigation, toolbars (most common)
- **32px** - For cards, larger buttons
- **40px** - For feature sections
- **48px** - For hero sections, large features

## Figma Export Instructions

1. **Select Section**: Choose the size you need (16, 20, 24, 32, 40, or 48px)
2. **Export Settings**:
   - Format: SVG
   - Color: Stroke only (no fill)
   - Stroke width: Defined per size in Figma
3. **Naming**: Icons follow kebab-case naming (e.g., `arrow-left`, `user-plus`)

## Implementation

These icons are from the **Feather Icons** library. You can:

1. **Use Feather Icons npm package**:
   ```bash
   npm install feather-icons
   ```

2. **Use React Feather**:
   ```bash
   npm install react-feather
   ```

3. **Manual SVG Export**: Export directly from your Figma file

## Color Variables

Icons use the following color variables from the design system:
- Primary: `#801C1E` (Deep Red)
- Neutral: `#111111` (Text Color)
- Stroke: `currentColor` (Inherits from parent)
